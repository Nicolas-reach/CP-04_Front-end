# CP-04_Front-end

# 🌿 EcoTrend — E-commerce Sustentável

Projeto desenvolvido para a disciplina de Front-end da FIAP (CP-04).

## 📋 Descrição

A EcoTrend é um e-commerce especializado em produtos sustentáveis e ecológicos, focado em promover um estilo de vida mais consciente. O site conta com páginas de navegação, listagem de produtos, detalhes e contato.

## 📄 Páginas

- **index.html** — Página inicial com banner, produtos em destaque e categorias
- **categorias.html** — Listagem de produtos com filtros laterais
- **detalhes.html** — Página de detalhes de um produto
- **contato.html** — Formulário de contato

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- Bootstrap 5
- Font Awesome 6
- Google Fonts (Playfair Display + DM Sans)
